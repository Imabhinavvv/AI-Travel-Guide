# 🌍 AI Travel Guide

AI Travel Guide is an AI-powered travel assistant that provides **smart travel recommendations** using the **Gemini API** and **Streamlit**. It helps users find the best travel options based on their preferences, including mode of transport, budget, and comfort.

## 🚀 Features
- 🔍 **AI-Powered Travel Suggestions** using **Google Gemini API**
- 🎨 **Interactive UI** built with **Streamlit**
- ✈️ **Multi-Mode Travel Options** (Train, Bus, Flight, Cab, Bike)
- 🔢 **Sort By** Price, Duration, or Departure Time
- 🗣️ **Supports Multiple Languages** (English, Hindi, Telugu, etc.)
- 🛠️ **Customizable Preferences** (Budget, Fastest, Most Comfortable)

## 🏗️ Tech Stack
- **Python**
- **Streamlit** (for UI)
- **LangChain** (for AI interactions)
- **Google Gemini API** (for travel recommendations)

## 📦 Installation
```bash
# Clone the repository
git clone https://github.com/Imabhinavvv/AI-Travel-Guide.git
cd AI-Travel-Guide

# Install dependencies
pip install -r requirements.txt

# Run the Streamlit app
streamlit run TripAI.py
```

## 🔑 Setup API Key
1. Get your **Google Gemini API key**.
2. Save it in a file named `genai.txt` inside the project directory.

## 🖼️ Usage
1. Enter **source** and **destination**.
2. Select **travel mode** and **preferences**.
3. Click **"Search Travel Options"** to get AI-powered recommendations.
